## 🐛 修复的问题

### 1. 设置页面没有退出机制 ✅

**问题描述**: 设置窗口打开后无法关闭，缺少关闭按钮。

**修复方案**: 
- 添加标题栏，右上角放置明显的关闭按钮（✕ 图标）
- 使用 `@Environment(\.dismiss)` 实现关闭功能

**修改文件**: `ShenMan/Views/SettingsView.swift`

```swift
private var titleBar: some View {
    HStack {
        Text("设置")
            .font(.shenManTitle2())
            .fontWeight(.bold)
        
        Spacer()
        
        // 关闭按钮
        Button(action: {
            dismiss()
        }) {
            Image(systemName: "xmark.circle.fill")
                .font(.title2)
                .foregroundColor(.secondary)
        }
        .buttonStyle(.plain)
        .help("关闭")
    }
}
```

### 3. 常规设置中自动检测与默认语言重复 ✅

**问题描述**: 
- "默认语言"选择器
- "自动检测语言"开关
- 两个功能概念重复，用户困惑

**修复方案**: 
- **移除"自动下载模型"开关**（简化设置）
- **移除"自动检测语言"开关**
- 在"默认语言"选择器中添加"自动检测"选项

**修改文件**: `ShenMan/Views/SettingsView.swift`

```swift
// 修复前 - 重复的设置
Section("模型") {
    Picker("默认模型", selection: $settings.selectedModel) { ... }
    Toggle("自动下载模型", isOn: $settings.autoDownloadModels)  // ❌ 移除
}

Section("语言") {
    Picker("默认语言", selection: $settings.defaultLanguage) { ... }
    Toggle("自动检测语言", isOn: $settings.enableLanguageDetection)  // ❌ 移除
}

// 修复后 - 简化的设置
Section("模型") {
    Picker("默认模型", selection: $settings.selectedModel) { ... }
}

Section("语言") {
    Picker("默认语言", selection: $settings.defaultLanguage) {
        Text("自动检测").tag(Language.chinese)  // ✅ 整合到一个选择器
        ForEach(Language.allCases.filter { $0 != .chinese }) { lang in
            Text(lang.displayName).tag(lang)
        }
    }
    .help("选择音频的默认语言，如果不确定请保持自动检测")
}
```

**新的语言选择器**:
```
默认语言 ▼
├── 自动检测  ← 新增选项
├── 中文
├── 英文
├── 日文
└── ...
```

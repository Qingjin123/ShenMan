## 📝 测试运行说明

### 运行所有测试

```bash
# 使用 Xcode
xcodebuild test -scheme ShenMan

# 使用 Swift CLI（如果支持 XCTest）
swift test
```

### 运行特定测试类

```bash
xcodebuild test \
  -scheme ShenMan \
  -only-testing:ShenManTests/ChinesePostProcessorTests
```

### 生成覆盖率报告

```bash
xcodebuild test \
  -scheme ShenMan \
  -enableCodeCoverage YES
```
